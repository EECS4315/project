Find the Main.java class that the group created within package "example".
Packages montecarlo, tictactoe and tree are from
 https://github.com/eugenp/tutorials/tree/950bbadc353bdca114befc98cf4a18476352220e/algorithms-searching/src/main/java/com/baeldung/algorithms/mcts

To run the application, please run "jpf Main.jpf", which exists either in /src or /bin.
Please note that the classpath and native_classpath values need to be updated with your personal directory.
